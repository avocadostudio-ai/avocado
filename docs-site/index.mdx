## What is AI Site Editor?

AI Site Editor is a chat-driven website editing platform. Site owners and content teams describe changes in natural language — the system applies them as validated, schema-checked operations with live preview, undo/redo, and plan approval.

**How it works**: a split-pane experience where your live website renders on the left and a chat interface on the right. Users type requests like _"add a testimonials section below the hero"_ or _"change the hero heading to Welcome"_, and the system plans and applies structured edits.

**For site owners**: connect the editor to your existing site by implementing a small API contract (component manifest + draft mode endpoints). No changes to your existing pages required.
